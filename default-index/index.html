（c++）数据结构初步（1）

发表于 2023-05-03 更新于 2023-05-18 分类于 C++ ， 数据结构

使用C++语言实现简单的线性表结构
值得一提的是，C++还有几种别的数据容器，比如map，vector等，这里也将一并介绍，与C语言重合度过高的地方我会将其简化，有明显差异的地方则会突出

  • 数组（插值，删除，改值，查值，遍历等）

    • C++中的数组定义同C一样容易理解，下面是一种常规的数组定义办法

        int num[10];

    • 虽然C语言的malloc类语句仍然可以使用，但是C++也拥有自己的特性，即用new来实现分配内存。同C语言类似，这种办法分配的内存需要使用delete进行释放。

        int *num = new int[10];
  delete []num;

    • 除此之外，C++还提供了两个容器，一个是array，一个是vector，要使用它们，需要引用头文件，并使用命名空间std。

        #include <array>
  #include <vector>
  using namespace std;

    • 下面是他们的初始化

        array1 <int, 10> values;//不对其值进行初始化
  array2 <int, 10> values {};//全部初始化为0
  array3 <int, 10> values {1, 2, 3};//前三个初始化为相应值，后面的初始化为0
  //array本质上仍然是静态数组，但其使用上更加安全可靠
  vector<int> v1;//size为0，空vector且未分配内存
  vector<int> v2(10);//已分配内存，且初始化为0
  vector<int> v3 = {1,2,3};//分配三个int内存，初始化为相应数值
  vector<int> v4 = v3;//等同于vector(int) v4(v3)，用另一个向量进行初始化
  vector<int> v5(10,1);//10个都初始化为1
  //vector类似于C语言的动态数组

    • 在初始化之外，作为一门面向对象的编程语言，array与vector还有许多高级成员函数，这里暂不讨论，以后需要使用时再讨论

（c）数据结构初步（1）

发表于 2023-05-02 更新于 2023-05-18 分类于 C语言 ， 数据结构

用C语言实现简单的线性表结构

  • 数组（插值，删除，改值，查值，遍历等）

    • C语言中，定义一个数组是很简单的。定义一个满足题意的静态数组可以用如下代码实现。 

        void DefineStaticArray(){
      int num[20];
      for(int i=0; i<20; i++){
          num[i] = i+1;
      }
  }

    • C语言中也可以使用malloc,calloc和realloc实现动态数组的效果，下面是代码实现。需要注意的是，这种方式定义的数组需要在程序末尾进行free操作释放被分配的内存。 

        void DefineDynamicArray(){
      int *num = NULL;
      num = (int*)malloc(sizeof(int));
      num = (int*)realloc(num, 20*sizeof(int));
      //或者采用下面这一行
      //num = (int*)calloc(20,sizeof(int));
      for(int i=0; i<20; i++){
          *(num+i) = i+1;
      }
  }

    • 也可以以下面一种形式定义柔性数组。这种数组结构要求一个结构体，结构体中数组成员必须处于任意其他数据类型的成员之后，且数组成员需要置于结构体末尾。定义完成后，使用sizeof(结构名)所返回的内存不包含数组内存，这种方法相比动态数组的优势在于访问效率更高一点。 

        void DefineFlexibleArray(){
      typedef struct FlexibleArray{
          int i;
          int num[];
      }FA;
      FA *fa = (FA*)malloc(sizeof(FA)+20*sizeof(int));
      fa->i = 20;
      for(int i=0;i<20;i++){
          fa->num[i] = i+1;
      }
  }

    • 插入和删除的操作本质上是一样的，都涉及到数组元素的位置移动，这里就只讨论插入值的操作。比如说我要将21这个数字插入第七个位置。 

        void InsertStaticArray(int *num, int *n_num){
      for(int i=0;i<6;i++){
          n_num[i] = num[i];
      }
      n_num[6]=21;
      for(int i=7;i<21;i++){
          n_num[i] = num[i-1];
      }
  }//提前定义n_num数组，输出时输出n_num

  void InsertDynamicArray(){
      num = (int*)realloc(num,21*sizeof(int));
      for(int i=20;i>6;i--){
          *(num+i) = *(num+i-1);
      }
      *(num+6) = 21;
  }

  void InsertFlexibleArray(){
      fa = (FA*)realloc(fa, sizeof(FA)+21*sizeof(int));
      for(int i=20;i>6;i--){
          fa->num[i] = fa->num[i-1];
      }
      fa->num[6] = 21;
  }

    • 在上面的代码中可以看出，静态数组的插入操作不够方便，动态数组和柔性数组所采用的方法基本类似。

    • 在知道需要更改的数组元素的索引的情况下可以直接对值进行更改，查值则只需要一个遍历操作就可实现，这里也不再赘述。

  • 栈（创栈，入栈，出栈，遍历，判断是否为空，判断是否满栈，清空栈）

    • 栈是一种操作受限的线性表，其是一种只允许在一端进行插入或删除的线性表，根据这个定义我们可以编写如下代码。这里定义这个栈有20个元素。（这里先不进行栈的拓展，只给出最基本的代码实现） 

        typedef struct SeqStaticStack{
      int body[20];
      int top;
  }Stack;

    • 这样定义的栈初始化很简单,将值清理为0 

        Stack *ps=(Stack*)malloc(sizeof(Stack));
  for(int i=0;i<20;i++){
      ps->body[i] = 0;
  }

    • 入栈

        int push(Stack *s, int i){
      if(s->top == 20)
          return 1;
      else{
          s->body[s->top] = i;
          s->top+=1;
          return 0;
      }
  }

    • 出栈

        int pop(Stack *s){
      if(s->top == 0)
          return 1;
      else{
          int i = s->body[s->top-1];
          s->body[s->top-1] = 0;
          s->top -=1;
          return i;
      }
  }

    • 遍历

        void travel(Stack *s){
      if(s->top == 0){
          printf("empty\n");
      }
      else{
          printf("from bottom:");
          for(int i=0;i<s->top;i++){
              printf("%d\t",s->body[i]);
          }
      }
  };

    • 判断是否为空或者为满或者清空在这里不做赘述

  • 链表（初始化，插值，删值，查改值，遍历，内存释放等）（这里以单链表做示范）

    • 链表的初始化包含三个部分，因为结构上链表包含一个头节点，一个尾节点和多个中间节点（也可以没有），每种节点均需要分配内存和初始化，同时节点内部指针指向不同，这里先定义一个结构体。

        typedef struct LinkedListNode{
      int num;
      struct LinkedListNode *NP;
  }LLN;

    • 接下来是初始化，函数功能为输入一个数字，返回一个存储该数据的，没有指向任何其他节点的节点指针。

        LLN *InitNode(int n){
      LLN *node;
      node = (LLN*)malloc(sizeof(LLN));
      node->num = n;
      node->next = NULL;
      return node;
  }

    • 接下来是插入节点，有三种插入方式，分别为头部插入，中间插入和尾部插入。

      • 头插函数需要两个参数，第一个是头节点的指针，第二个是需要插入的节点的指针。函数返回整数。

          int HeadInsert(LLN *OHead,LLN *NHead){
      NHead->next = OHead;
      if(NHead->next == NULL){
          printf("Failed,head node is still old one\n");
          return 1;
      }
      printf("Success,head node is new one\n");
      return 0;
  }

      • 尾插函数需要两个参数，一个是头节点的指针，第二个是需要插入的节点的指针。函数无返回值。
        void TailInsert(LLN *Head,LLN *Tail){
        LLN *ptr = Head;
        while(ptr->next!=NULL){
        ptr = ptr->next;
        }
        ptr->next = Tail;
        }

      • 插入函数将把节点插入到原节点后方，需要两个参数，一个是原节点，一个是插入的节点。函数无返回值，该函数在知晓尾节点指针时也可直接做尾插操作。

          void Insert(LLN *f,LLN *n){
      n->next = f->next;
      f->next = n;
  }

    • 删除函数要求传入链表头节点和一个需要从表中删除的数据，无返回值。注意该函数没有将节点free，也就是说其相当于将节点从链表中移除了。

        void Delete(LLN *head, int num){
      LLN *ptr = head;
      LLN *pt = NULL;
      while(1){
          if(ptr->num!=num){
              pt = ptr;
              ptr = ptr->next;
          }
          else break;
      }
      if(pt!=NULL)
          pt->next = ptr->next;
      ptr->next = NULL;
  }

    • 查改值和遍历与上面类似，这里贴一个遍历，输出传入节点之后的所有节点值，当节点为头时输出链表内值。

        void travel(LLN *head){
      LLN *ptr = head;
      while(ptr->next!=NULL){
          printf("%d\t",ptr->num);
          ptr = ptr->next;
      }
      printf("%d\n",ptr->num);
  }

  • 队列（初始化，入队，出队，遍历等，和栈基本类似）

    • 使用数组来写队列未免太过绿色，这里采用链表的办法来理解列表，事实上代码都在链表节中实现了，只需要限定哪一端出哪一端入即可，复杂的讨论不在这里进行。

数据结构（1）

发表于 2023-05-01 更新于 2023-05-18 分类于 小 实 践 ， 数据结构

本篇博客讨论的是数据结构中的线性表

  • 线性表的定义

    • 线性表是数据结构的一种，一个线性表指的是零到n个数据元素的有限序列，其中的元素都有一个确定的位置，数据元素之间是一对一的关系。线性与否的判别标准是逻辑层次上的，而非存储层次上的。
    • 线性表具有下面几种结构特点：
      1. 均匀性：虽然不同数据表的数据元素可以是各种各样的，但对于同一线性表的各数据元素必定具有相同的数据类型和长度。
      2. 有序性：各数据元素在线性表中的位置只取决于它们的序号，数据元素之前的相对位置是线性的，即存在唯一的“第一个“和“最后一个”的数据元素，除了第一个和最后一个外，其它元素前面均只有一个数据元素(直接前驱)和后面均只有一个数据元素（直接后继）。

  • 线性表的分类

    • 数组


      数组是最基础的一种数据结构，他是一种线性表数据结构，用一组连续的内存空间，来存储一组具有相同类型的数据。数组是用连续内存空间存储相同类型的元素，就是因为有这个限制条件，使得数组按照下标随机访问(随机访问：可以用同等的时间访问到一组数据中的任意一个元素)数组中数据元素时间复杂度达到 O(1) 级别。当然这样的限制也有缺点，在头部或者中间进行数据删除、插入操作时，为了保证这个连续性，需要对数据进行大量的复制迁移来保持此特性。
      这里有一个注意点，我们是通过数组下标访问数据时，时间复杂度才是 O(1),当我们通过数据查找元素时，我们需要遍历数组查找对应的数据，时间复杂度是 O(n)。
      数组是一个固定长度的数据结构，这是它的一个巨大弱点，即需要在定义中声明数组的长度，装满后就装不进去了。在高级编程语言中，往往都使用动态数组来解决这个问题，所谓动态数组就是自动会进行数组扩容，当数组容量达到某个临界点的时候，动态数组会开辟一个更大的数组，然后将原来的元素复制过去。动态数组在帮你自动扩容的同时，也带来了复制元素的代价：
      • 当数组需要扩容时，需要进行元素复制。
      • 当删除数组元素时也需要进行元素复制，因为当一删除某一个元素后，数组中间就会有一个空位置，需要把后面的元素都往前挪一格。

    • 栈


      栈是一种操作受限的线性表，其是一种只允许在一端进行插入或删除的线性表，下面是一些涉及栈的概念
      • 栈顶（Top）：线性表允许进行插入删除的那一端。
      • 栈底（Bottom）：固定的，不允许进行插入和删除的另一端。
      • 空栈：不含任何元素的空表。
        栈又称为后进先出（Last In First Out）的线性表，简称LIFO结构

    • 队列


      队列是一种操作受限的线性表，其是只允许在一端进行插入操作，而在另一端进行删除操作的线性表，下面是一些涉及队列的概念
      • 队头（Front）：允许删除的一端，又称队首。
      • 队尾（Rear）：允许插入的一端。
      • 空队列：不包含任何元素的空表。
        队列是一种先进先出（First In First Out）的线性表，简称FIFO。允许插入的一端称为队尾，允许删除的一端称为队头。

    • 链表


      数组的元素地址是强制连续的，但是链表的元素内存地址可以是不连续的 ，它通过地址引用的方式指向下一个元素的位置，所以链表的数据结构看起来比较像一个串。
      对于单链表而言，它的数据结构注定了它无法快速的访问某个元素，只能依靠遍历的方式慢慢查找，所以说链表的查询数据是比较慢的，是O(N) 这个级别，但是它在头节点（第一个节点）进行插入和删除比较快，只需要O(1) 就可以了，因为你只需要改变一下第一个元素的引用。

  • 附录：大O表示法：

    • O(1) ：常数级别，无论输入多大，它执行的步数都是恒定的，不会因为输入变大而变大，哈希表的查找就是这个级别。
    • O(N) ：线性级别，随着输入变大耗费的步数也正向相关，遍历算法就是这个级别。
    • O(logN) ：对数级别，输入每变大一倍，耗费步骤则增加1，二分查找算法属于这个级别。
    • O(N²) ：平方级别，随着输入的变大所耗费的步数会成倍增加，一般当你的算法使用双层for循环就是这个级别，比如冒泡排序。

HTML初识（1）

发表于 2023-04-30 更新于 2023-05-03 分类于 HTML ， 基础

HTML，全称超文本标记语言。本来我不想学这个的，但是为了能看得懂爬虫内容，就稍微涉猎一下这方面前端的内容吧，学会之后就可以自己修改自己博客的前端样式了，想想都刺激。

  • HTML元素

    • HTML 元素以开始标签起始

    • HTML 元素以结束标签终止

    • 元素的内容是开始标签与结束标签之间的内容

    • 某些 HTML 元素具有空内容（empty content）

    • 空元素在开始标签中进行关闭（以开始标签的结束而结束）

    • 大多数 HTML 元素可拥有属性

    • 代码示例： 

        <html>
  <body>
  <p>这是一个段落</p>
  </body>
  </html>

    • 以上代码包含三个html元素，这三个元素均拥有两个标签标记元素的开始和结束。在语句<p>这是一个段落</p>中<p>作为开始标签，<\p>作为结束标签，中间夹的部分即为该元素的内容。并非所有HTML元素都拥有结束标签

    • 以上代码为三个html元素的嵌套，一般html文档由多个嵌套的html元素组成

  • HTML空元素

    • 没有内容的 HTML 元素被称为空元素。空元素是在开始标签中关闭的。
    • <br> 就是没有关闭标签的空元素（<br> 标签定义换行）。
    • 在 XHTML、XML 以及未来版本的 HTML 中，所有元素都必须被关闭。
    • 在开始标签中添加斜杠，比如 <br />，是关闭空元素的正确方法，HTML、XHTML 和 XML 都接受这种方式。
    • 初学还是能关就关，符合规范

  • HTML大小写

    • HTML大小写不敏感，但是建议还是用小写
